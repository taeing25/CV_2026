## 1. 코드 설명

`01.py`는 MNIST 데이터를 정규화해 간단한 완전연결 신경망으로 학습하고, 테스트 성능과 샘플 예측 결과를 콘솔로 확인하는 코드입니다.

### 1-1. 라이브러리 임포트

- `numpy`: 예측 확률 중 최댓값 인덱스를 구해 최종 클래스(숫자)를 얻을 때 사용합니다.
- `tensorflow`: 데이터셋 로드, 모델 구성, 학습/평가에 사용합니다.

코드의 핵심 동작은 TensorFlow/Keras 안에서 이루어집니다.

### 1-2. `load_and_prepare_data()`

이 함수는 MNIST 데이터 로드와 전처리를 담당합니다.

```python
(x_train, y_train), (x_test, y_test) = tf.keras.datasets.mnist.load_data()
```

MNIST는 기본적으로 학습/테스트 세트가 나뉘어 제공됩니다.

- 학습 이미지: 60,000장
- 테스트 이미지: 10,000장
- 이미지 크기: 28x28 흑백

이후 정규화를 수행합니다.

```python
x_train = x_train.astype("float32") / 255.0
x_test = x_test.astype("float32") / 255.0
```

픽셀 값(0~255)을 0~1 범위로 바꿔 학습 안정성을 높입니다.

함수 반환값:

- `x_train`, `y_train`, `x_test`, `y_test`

### 1-3. `build_model()`

이 함수는 분류 모델을 생성하고 컴파일합니다.

모델 구조:

1. `Flatten(input_shape=(28, 28))`
2. `Dense(128, activation="relu")`
3. `Dense(10, activation="softmax")`

설명:

- `Flatten`: 2차원 이미지를 1차원 벡터로 펼칩니다.
- `Dense(128, relu)`: 숫자 특징을 학습하는 은닉층입니다.
- `Dense(10, softmax)`: 0~9 클래스 확률을 출력합니다.

컴파일 설정:

- 옵티마이저: `adam`
- 손실함수: `sparse_categorical_crossentropy`
- 평가지표: `accuracy`

`sparse_categorical_crossentropy`를 사용했기 때문에 레이블을 원-핫 인코딩하지 않고 정수 그대로 사용할 수 있습니다.

### 1-4. `main()`

실행의 메인 흐름을 담당합니다.

1. 데이터 로드 및 전처리
2. 모델 생성
3. 모델 학습
4. 테스트 데이터 평가
5. 샘플 1장 예측
6. 학습/검증 정확도 출력

학습 호출:

```python
history = model.fit(
	x_train,
	y_train,
	epochs=5,
	batch_size=32,
	validation_split=0.1,
	verbose=1,
)
```

주요 파라미터:

- `epochs=5`: 전체 데이터를 5번 반복 학습
- `batch_size=32`: 한 번에 32개 샘플 처리
- `validation_split=0.1`: 학습 데이터의 10%를 검증에 사용

평가 및 예측:

- `model.evaluate(...)`로 테스트 손실/정확도 계산
- `model.predict(...)` + `np.argmax(...)`로 샘플 이미지의 예측 숫자 계산

## 2. 전체 흐름

`01.py`는 아래 순서로 실행됩니다.

1. MNIST 데이터를 불러옵니다.
2. 이미지를 정규화합니다.
3. 완전연결 신경망(Flatten + Dense + Dense)을 구성합니다.
4. 모델을 학습합니다.
5. 테스트 데이터로 성능을 확인합니다.
6. 테스트 이미지 1장을 예측합니다.
7. 마지막 epoch의 학습/검증 정확도를 출력합니다.

정리하면, 앞부분은 데이터 준비, 중간은 모델 학습, 뒷부분은 평가와 예측 확인입니다.

## 3. 개념 설명

### MNIST

손글씨 숫자(0~9) 분류를 위한 대표적인 입문용 데이터셋입니다.

### 정규화

입력값 범위를 맞춰 학습을 안정화하는 과정입니다. 여기서는 0~255를 0~1로 변환합니다.

### Flatten

2차원 이미지 데이터를 1차원 벡터로 펼쳐 Dense 레이어에 입력할 수 있게 합니다.

### Dense 레이어

완전연결층으로, 입력 특징을 조합해 분류 경계를 학습합니다.

### ReLU

은닉층 활성화 함수입니다. 비선형성을 부여해 복잡한 패턴 학습을 가능하게 합니다.

### Softmax

출력층에서 각 클래스의 확률을 계산합니다. 가장 큰 확률의 클래스가 최종 예측입니다.

### sparse_categorical_crossentropy

정수 레이블을 직접 사용하는 다중분류 손실함수입니다.

## 4. 결과물 설명

이 스크립트는 파일을 저장하기보다 **콘솔 출력**으로 결과를 제공합니다.

출력 항목:

- 테스트 손실
- 테스트 정확도
- 샘플의 실제 레이블
- 샘플의 예측 레이블
- 마지막 epoch 학습 정확도
- 마지막 epoch 검증 정확도

예시 형태:

```text
학습 완료
테스트 손실: 0.0843
테스트 정확도: 0.9729
실제 레이블: 7
예측 레이블: 7

학습 기록 예시:
- 마지막 epoch 학습 정확도: 0.9852
- 마지막 epoch 검증 정확도: 0.9767
```

해석 포인트:

- 테스트 정확도가 높을수록 일반화 성능이 좋습니다.
- 학습 정확도와 검증 정확도의 차이가 지나치게 크면 과적합을 의심할 수 있습니다.
- 샘플 예측이 실제 레이블과 일치하는지로 모델 동작을 직관적으로 확인할 수 있습니다.
