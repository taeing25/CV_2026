# 이미지 처리 코드 설명 - 원본 vs 그레이스케일 비교

## 개요

원본 이미지(컬러)와 같은 이미지의 흑백(그레이스케일) 버전을 나란히 보여주는 프로그램입니다.

마치 "원본 사진" vs "흑백 사진" 을 한 화면에 비교하는 것과 같습니다.

---

## 필요한 라이브러리

```python
import cv2 as cv
import numpy as np
import sys
```

**라이브러리 설명:**
- `cv2 as cv` : OpenCV(컴퓨터 비전 라이브러리) - 이미지를 다루는 여러 도구들의 모음
- `numpy as np` : NumPy - 숫자를 다루는 도구 (여러 이미지를 한 번에 처리할 때 유용)
- `sys` : 시스템 관련 도구 - 프로그램에 문제가 생기면 종료할 때 사용

---

## 1단계: 이미지 파일 열기

```python
img = cv.imread('soccer.jpg')
```

**하는 일:**
- `cv.imread()` : 이 그림을 읽어주는 명령어
- `'soccer.jpg'` : 같은 폴더에 있는 "soccer.jpg" 파일을 찾음
- `img` : 메모리에 저장된 이미지 (변수 이름)

컴퓨터의 저장 공간에서 사진 파일을 불러와서 메모리에 올리는 것입니다.

---

## 2단계: 파일 확인하기

```python
if img is None:
    sys.exit('파일을 찾을 수 없습니다. 파일명과 경로를 확인하세요.')
```

**하는 일:**
- `if img is None:` : 만약에 이미지가 제대로 열리지 않았다면 실행
- `sys.exit()` : 프로그램을 멈추고 사용자에게 메시지를 보여줌

이미지 파일이 제대로 열렸는지 확인하는 안전장치입니다.
(예: 파일이 다른 폴더에 있거나 없으면 알려줌)

---

## 3단계: 이미지 크기 줄이기

```python
img = cv.resize(img, dsize=(0,0), fx=0.5, fy=0.5)
```

**하는 일:**
- `cv.resize()` : 이미지 크기를 바꿔주는 명령어
- `dsize=(0,0)` : 자동으로 계산하도록 설정
- `fx=0.5, fy=0.5` : 가로 세로 모두 50%로 줄임 (절반 크기)

큰 사진을 작은 사진으로 축소합니다. 화면에 표시하기 편하도록 또는 처리 속도를 빠르게 하기 위함입니다.

---

## 4단계: 컬러 이미지를 흑백으로 변환

```python
gray = cv.cvtColor(img, cv.COLOR_BGR2GRAY)
```

**하는 일:**
- `cv.cvtColor()` : 색깔을 바꿔주는 명령어
- `cv.COLOR_BGR2GRAY` : 파란색, 초록색, 빨간색(BGR)을 회색(GRAY)으로 변환

컬러 사진을 흑백 사진으로 바꾸는 것입니다. 마치 흑백 필터를 씌운 것처럼요.

---

## 5단계: 흑백 이미지를 다시 컬러 형식으로 변환

```python
gray_3channel = cv.cvtColor(gray, cv.COLOR_GRAY2BGR)
```

**하는 일:**
- `gray` : 흑백 이미지 (1개 색깔 채널)
- `cv.COLOR_GRAY2BGR` : 흑백(1개 채널)을 컬러 형식(3개 채널)으로 변환

**왜 이렇게 하나요?**

흑백 이미지와 컬러 이미지는 내부 구조가 다릅니다. 다음 단계에서 두 이미지를 붙이려면 같은 구조여야 하므로, 흑백 이미지를 "형식상으로" 컬러 형식으로 변환합니다. 색상은 여전히 회색입니다.

---

## 6단계: 두 이미지를 가로로 붙이기

```python
combined = np.hstack((img, gray_3channel))
```

**하는 일:**
- `np.hstack()` : 이 두 이미지를 가로로 붙여주는 명령어
- `(img, gray_3channel)` : 원본 이미지와 흑백 이미지

두 사진을 나란히 이어 붙이는 것입니다.
```
[원본 컬러 이미지] [흑백 이미지]
```

---

## 7단계: 결과를 화면에 보여주기

```python
cv.imshow('Original vs Grayscale', combined)
```

**하는 일:**
- `cv.imshow()` : 이미지를 화면에 띄워주는 명령어
- `'Original vs Grayscale'` : 창의 제목
- `combined` : 붙여진 이미지

만든 결과물을 새로운 창에서 보여줍니다.

---

## 8단계: 아무 키나 누를 때까지 기다리기

```python
cv.waitKey(0)
```

**하는 일:**
- `cv.waitKey()` : 아무 키나 누를 때까지 기다려주는 명령어
- `(0)` : 무한정 기다림 (0은 "계속 기다려"라는 의미)

사용자가 아무 키나 누를 때까지 이미지를 화면에 유지합니다.

---

## 9단계: 모든 창 닫기

```python
cv.destroyAllWindows()
```

**하는 일:**
- `cv.destroyAllWindows()` : 모든 이미지 창을 닫아주는 명령어

사용자가 키를 누르면 열려있던 이미지 창을 모두 닫습니다.

## 전체 흐름 요약

3. 큰 사진을 작은 사진으로 줄이기

9. 창 닫기

---

## 용어 설명

| 용어 | 의미 |
|------|------|
| 이미지 | 사진이나 그림 파일 |
| 그레이스케일 | 흑백 이미지 (회색 톤으로 표현) |
| BGR | 컴퓨터가 이미지를 표현하는 방식 (파란색, 초록색, 빨간색) |
| 채널 | 이미지의 색깔 정보 (컬러=3개, 흑백=1개) |
| 픽셀 | 이미지를 이루는 가장 작은 점 |
